##  使用方法

### 1. 查看研究报告
```bash
# 直接查看研究报告
cat output/ai_research_2025/research.md

# 或者在浏览器中打开
open output/ai_research_2025/research.md
```

### 2. 查看幻灯片
```bash
# 打开导航页面
open output/ai_research_2025/index.html

# 或者直接查看HTML幻灯片
open output/ai_research_2025/pages/page-1.pptx.html
```

### 3. 使用PPTX文件
```bash
# 完整版PPTX
open output/ai_research_2025/ai_research_2025.pptx

# 摘要版PPTX
open output/ai_research_2025/ai_research_summary_2025.pptx
```

### 4. 重新生成PPTX
```bash
# 进入项目目录
cd output/ai_research_2025

# 运行生成脚本
python3 generate_pptx.py
```
