## 功能：
1. computer use工具集（windows/mac优先支持，linux/ubuntu后续支持） 
2. 统一网关，架构收编（外部交互统一，飞书/微信/office claw）
3. 前端界面优化（流式消息同步，常用功能首页展示，包括skill，gateway）
4. 在ubuntu配置多个机器人 
5. 云端部署自动化    
6. github action实现按新建tag出包
7. agent hook功能（对标claude code）   
8. agent plugin支持（对标claude code） 
9. agent测评增强（增加skill测评） 
10. 接口入参强校验（修改，删除场景）  
11. agent runtime，大模型报错（chat->response），恢复/自愈能力提升
12. 多模态（openai，glm，deepseek，kimi，minimax）
13. office claw走ACP访问agent teams不支持传图和文件 
14. 检索（历史记忆），排序（BM25未应用到mcp和memory） 
15. 普通模式支持subagent （参考opencode），可后台运行    
16. shell支持后台运行 （参考codex） 
17. 外部资源统一管理（clawhub， gh） ，提供skillhub运行时加载  
18. 模型认证（provider，model）支持oauth（参考opencode） 
19. codehub cli（对标gh）支持

## 长期规划：
1. 群聊：1.支持在agent teams中会话群聊，也支持在现有会话临时群聊；2.支持飞书中多bot协作和群聊

## 场景支持：
1. AI简报（机器人自动生成简报并发布JX，提供JX CLI）
2. 多bot协作：洞察bot（提feature），项目分析bot（项目管理，排期，合并issue），开发bot（代码开发，提PR），检视bot（检视PR并合并）
